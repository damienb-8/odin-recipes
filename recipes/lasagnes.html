Lasagnes

Accueil

Description

Voici une recette de lasagnes facile que tout le monde peut réaliser !

Ingrédients

  • Un ordinateur
  • Des chaussures
  • Un four
  • C'est tout !

Étapes

  1. Allumez votre ordinateur.
  2. Commandez une barquette de lasagnes surgelés.
  3. En attendant le livreur, continuez The Odin Project.
  4. Mettez vos chaussures pour aller chercher le colis quand vous entendez la sonnette.
  5. Faites cuire les lasagnes comme indiqué sur l’emballage.
  6. Mangez les lasagnes.
  7. C'est bon, vous pouvez arrêter de lire, c'est fini !
